# Smart Diffuser (ESP32 + Home Assistant)

Convert a standard diffuser into a fully local smart device using ESP32 and ESPHome.

## Features

* No cloud required
* Works with Home Assistant
* Simulates physical button presses
* Fully internal wiring (no external devices)

## Hardware

* ESP32
* Buck converter (24V → 5V)
* Asakuki 500ml diffuser

## Status

Working build. Second unit will be used for full documentation.

## Notes

This diffuser uses button cycling (state machine), not direct state control.
